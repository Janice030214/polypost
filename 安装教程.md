# Polypost 零基础安装教程

> 这份教程假设你**从来没用过终端、没装过 Node.js、也没碰过命令行**。
> 一步一步照做，半小时之内就能用上。

---

## 总览：你要做的 8 件事

```
1️⃣  装 Node.js          （程序运行环境，就像 PDF 文件需要 PDF 阅读器一样）
2️⃣  下载 Polypost 代码  （从 GitHub 下一个压缩包）
3️⃣  打开「终端」        （Mac 和 Windows 的方式不一样）
4️⃣  进入项目文件夹       （在终端里）
5️⃣  装项目依赖           （一行命令：`npm install`）
6️⃣  填配置文件 .env     （Strapi 的地址、账号、token 等）
7️⃣  启动 Polypost       （`npm start`）
8️⃣  在浏览器里打开      （http://localhost:4000）
```

下面每一步都细写。**对应你的系统选 Mac 或 Windows 的版本看**。

---

## 1️⃣ 装 Node.js（程序运行环境）

Polypost 是用 JavaScript 写的，需要 **Node.js** 才能跑。

### Mac

1. 打开浏览器，去 → https://nodejs.org/
2. 主页中央两个绿色大按钮，点 **「LTS」**（左边那个，写着 "Recommended For Most Users"）
3. 下载完了打开下载文件夹，**双击** 那个 `.pkg` 文件
4. 安装窗口里一路点 **「继续」→「同意」→「安装」**，最后可能要输 Mac 密码
5. 看到 ✅ 成功的勾就关掉

### Windows

1. 打开浏览器，去 → https://nodejs.org/
2. 主页中央，点 **「LTS」**（左边那个）
3. 下载完了双击 `.msi` 文件
4. 安装窗口一路 **Next → I accept → Next → Next → Install**，需要管理员授权点 **「是」**
5. 最后 **Finish**

### 验证装好了

我们稍后会在终端里输 `node --version`，会显示 `v22.x.x` 之类的版本号——看到就说明装好了。

---

## 2️⃣ 下载 Polypost 代码

最简单的方法：**下载 zip 压缩包**（不用安装 git）。

1. 打开 → https://github.com/Janice030214/polypost
2. 找到右上方那个**绿色的 `<> Code` 按钮**，点一下
3. 弹出的小窗口里，**最下面**有一行 **「Download ZIP」**，点它
4. 浏览器开始下载一个文件叫 `polypost-main.zip`
5. **解压**：
   - **Mac**：双击 zip 文件，自动会解出一个 `polypost-main` 文件夹
   - **Windows**：右键 zip → 「全部解压」→ 选个位置
6. 把解出来的 **`polypost-main` 文件夹** 拖到一个好找的地方，比如：
   - Mac → 拖到 **桌面**
   - Windows → 拖到 **桌面**

> 📌 记住这个文件夹的位置，后面要用。

---

## 3️⃣ 打开「终端」

### Mac

方法 A（推荐，最快）：
1. 同时按 **`Command + 空格`** （键盘最下面那个 ⌘ 键 + 空格键）
2. 屏幕中央会弹出一个搜索框（Spotlight）
3. 输入 **`终端`** 或者 **`terminal`**
4. 看到一个**黑色的方框图标**叫「终端」，**按回车**
5. 一个全黑（或白）的窗口打开了，里面有一行字以你的电脑名结尾，光标在闪——这就是终端

### Windows

方法 A（推荐，Win 10/11 都行）：
1. 同时按 **`Win + R`**（Win 是键盘上有 Windows 图标那个键）
2. 弹出小窗口，输入 **`cmd`**，按回车
3. 一个黑色窗口打开了——这就是终端（叫"命令提示符"）

或者：
- 直接点屏幕左下角**搜索框**，输 `cmd`，回车

> 终端就是这个工具——**电脑里所有"命令"都在这里跑**。后面我们要在它里面输几行命令。

---

## 4️⃣ 在终端里"进入"项目文件夹

打开了终端，光标在闪。我们需要"切换"到 Polypost 文件夹里。命令是 `cd`（change directory，切换目录）。

### 最简单的办法：拖文件夹

#### Mac

1. 在终端里**先输入** `cd `（注意 `cd` 后面有一个**空格**！别按回车！）

   ```
   cd 
   ```

2. 然后**用鼠标把 `polypost-main` 文件夹拖到终端窗口里**——文件夹的路径会自动出现在 `cd ` 后面
3. 按**回车**
4. 现在终端的提示符里会出现 `polypost-main`，表示你已经在文件夹里了

#### Windows

1. 在终端里**先输入** `cd `（注意有一个**空格**！别按回车！）
2. **用鼠标把 `polypost-main` 文件夹拖到终端窗口里**——文件夹的路径会自动出现
3. 按**回车**

### 验证你进对了

输入这一行，按回车：

```bash
ls
```

（Windows 用 `dir` 也行，不过 `ls` 在新版 Windows 也能用）

你应该看到这些文件名：
```
LICENSE   README.md   package.json   public   server.js   src   安装教程.md
```

看到了就对了 ✅

---

## 5️⃣ 装项目依赖（一行命令）

依赖 = 项目用到的"外部代码包"。Polypost 用了 `express` 和 `node-html-markdown` 两个包，需要先下载。

在刚才那个终端窗口里，输入：

```bash
npm install
```

按回车，开始下载。会看到一堆滚动的字、进度条之类的，正常。

**等大概 30-60 秒**，下载完会回到提示符等你输下一条命令。

> 如果提示 `npm: command not found`，说明 Node.js 没装好或者你**需要关掉终端重开一个新的**（让 Node.js 的路径生效）。

---

## 6️⃣ 填配置文件 .env（最关键的一步）

### 第 1 小步：复制模板

项目里有一个文件叫 **`.env.example`**，是一份**配置文件模板**。我们要把它复制一份，改名叫 **`.env`**，然后往里填真实的值。

#### Mac

在终端里输：

```bash
cp .env.example .env
```

按回车。没报错就行（不会显示任何东西）。

#### Windows

```bash
copy .env.example .env
```

按回车。

### 第 2 小步：用「文本编辑器」打开 .env

#### Mac

在终端里输：

```bash
open -a TextEdit .env
```

按回车。**「文本编辑」**应用会自动打开 `.env` 文件，里面会有这样的内容：

```env
# ===== Strapi connection =====
STRAPI_URL=https://your-strapi.example.com
STRAPI_TOKEN=your_strapi_api_token

# ===== Strapi admin login =====
STRAPI_ADMIN_EMAIL=you@example.com
STRAPI_ADMIN_PASSWORD=your_admin_password
...
```

#### Windows

```bash
notepad .env
```

按回车。**记事本**会打开。

### 第 3 小步：把右边的值替换成你的真实信息

下面每一项**逐条**改：

#### a) STRAPI_URL（你的 Strapi 后台地址）

把这一行：
```
STRAPI_URL=https://your-strapi.example.com
```
改成你 Strapi 后台的**主域名**（**不带 `/admin`**），例如：
```
STRAPI_URL=https://strapi-admin.atlascloud.ai
```

#### b) STRAPI_TOKEN（API 令牌）

**怎么拿到这个 token**：

1. 浏览器登录你的 Strapi 后台（例如 `https://strapi-admin.atlascloud.ai/admin`）
2. 左下角点你的头像 → 或者直接进 **Settings**（小齿轮图标）
3. 左侧菜单找 **「API Tokens」**
4. 右上角点 **「Create new API Token」**
5. 填写：
   - **Name**：`Polypost`（随便）
   - **Token duration**：选 `Unlimited`（不过期）
   - **Token type**：选 **`Full access`**
6. 点 **「Save」**
7. **保存后**，页面顶部会出现一长串字符（很长，几百个字符），**只显示这一次**！
8. **立刻复制**这一长串（点旁边的复制图标）
9. 回到记事本，把
   ```
   STRAPI_TOKEN=your_strapi_api_token
   ```
   等号后面替换成你刚才复制的那一长串。

> ⚠️ 这串 token **只显示一次**，关了页面就找不回了。如果忘了复制，删掉重新创建一个就行。

#### c) STRAPI_ADMIN_EMAIL 和 STRAPI_ADMIN_PASSWORD

填**你登录 Strapi 后台用的邮箱和密码**：

```
STRAPI_ADMIN_EMAIL=你的邮箱@example.com
STRAPI_ADMIN_PASSWORD=你的后台登录密码
```

> 这个是 Polypost 用来自动翻译（调 Strapi 后台那个 Translate with AI 插件）+ 一键发布用的。

#### d) VISION_*（图片自动写 alt，可选）

如果你想让 AI 自动给文章图片写 alt 描述，填这三项。不填就空着，alt 留给你手动填。

- `VISION_API_BASE`：模型接口地址，例如 `https://api.atlascloud.ai/v1`
- `VISION_API_KEY`：你的模型 API key
- `VISION_MODEL`：模型名，推荐 `qwen/qwen3-vl-30b-a3b-instruct`

> 也可以稍后在网页里点 **⚙️ 视觉模型设置** 按钮来配，效果一样。

#### e) REVIEW_*（点预览时 AI 审稿，可选）

跟视觉模型一样的填法，只是模型名换成 `deepseek-ai/deepseek-v4-pro`（默认）或别的文本模型。不填就空着。

### 第 4 小步：保存

- **Mac TextEdit**：菜单栏 → 文件 → 存储（或 `Command + S`）→ **关掉这个窗口**
- **Windows 记事本**：菜单栏 → 文件 → 保存（或 `Ctrl + S`）→ **关掉这个窗口**

> 📌 保存好后，**回到终端窗口**（点终端窗口让它在前面）。

---

## 7️⃣ 启动 Polypost

在终端里输：

```bash
npm start
```

按回车。看到这两行字就说明成功了：

```
Polypost 上传器已启动：http://localhost:4000
```

> ⚠️ **不要关掉这个终端窗口！** 只要 Polypost 在跑，它就一直要打开。关了终端 Polypost 就停了。

---

## 8️⃣ 在浏览器里打开

打开浏览器（**Chrome、Safari、Edge 都行**），在地址栏输：

```
http://localhost:4000
```

按回车。

你会看到一个**淡紫色**的网页，顶部标题是 **「Strapi 博客上传 `</>` 」**——成功！

---

## 之后每次怎么用

**装好之后，以后用 Polypost 只需要 3 步**：

1. 打开终端
2. 进项目文件夹（拖文件夹到终端那一招）
3. 输 `npm start` 回车
4. 浏览器开 `http://localhost:4000`

## ❓ 常见问题

### Q1：终端里命令报 `command not found: npm`

A：Node.js 没装好，或者你**装 Node.js 前**就开着的终端没用上新的环境变量。
- **解决**：**完全关掉终端**（Mac 是 `Command + Q`，Windows 直接关窗口），**重新打开**，再 `npm install` / `npm start`。

### Q2：浏览器打开 `http://localhost:4000` 显示「无法访问此网站」

A：Polypost 没在跑。
- **解决**：去终端看，是不是终端窗口被你关了？或者终端里有红色报错？把终端窗口翻回来，再输 `npm start`。

### Q3：网页能打开，但点处理文章报错

A：**`.env` 配置有问题**。
- 最常见的：`STRAPI_TOKEN` 填错、过期，或者后台 URL 不对。
- **解决**：去终端按 `Ctrl + C`（这能停掉正在跑的 Polypost），改 `.env` 文件，再 `npm start`。

### Q4：`.env` 文件在 Mac 上隐藏了找不到

A：以 **`.`** 开头的文件 Mac 默认隐藏。
- **解决**：在 Finder 里按 **`Command + Shift + .`**（点是英文句号），隐藏文件就出现了。再按一次就藏回去。

### Q5：我能不能用别人的电脑/手机/iPad 远程访问 Polypost？

A：**不能**。Polypost 跑在 `localhost`，只有这台电脑上能访问。这是设计上的——它装着你的 Strapi 密码，不应该暴露到公网。

### Q6：电脑重启了，Polypost 怎么再启动？

A：每次重启电脑后，要**重新启动**：
1. 打开终端
2. 拖文件夹进去 cd
3. 输 `npm start`
4. 浏览器开 `http://localhost:4000`

（不用再 `npm install`，那是一次性的）

### Q7：怎么彻底关掉 Polypost？

A：在跑 Polypost 那个终端窗口里，**按 `Ctrl + C`**（Mac 也是 Ctrl，不是 Command）。或者直接关掉终端窗口。

### Q8：能不能让 Polypost 开机自动启动？

A：可以但需要额外配置，超出新手范围。先按上面的方法用着就行。
